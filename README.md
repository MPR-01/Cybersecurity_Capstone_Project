# Cyber Warrior Program | DeepCytes Cyber Labs (UK)

## Unified Cyber Resource Intelligence Platform (UCRIP)

An **AI-powered, jurisdiction-sensitive platform** designed to assist with cybersecurity incident guidance and reporting.

---

## Key Features

### Region-Aware Intelligence
Switch between **Finland** and the **USA** to access localized cybersecurity resources and reporting guidance.

### State-Level Compliance
Automatically detects **U.S. state-specific reporting mandates** (e.g., **Virginia’s 24-hour reporting rule**) and provides relevant guidance.

### Incident Reporting Wizard
A **visual classification tool** that helps users quickly determine the correct **official reporting portal** for a cybersecurity incident.

### AI Guidance Assistant
A **Gemini-powered chatbot** capable of providing **natural language incident assessment and guidance**.

### Privacy First
Stateless architecture with built-in privacy features:
- **Clear Chat**
- **Export Transcript**
